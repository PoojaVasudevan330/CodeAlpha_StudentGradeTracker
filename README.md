# CodeAlpha - Student Grade Tracker

## 📌 Project Overview
This is a Java-based Student Grade Tracker application developed as part of the CodeAlpha internship program. The project helps manage student records, calculate grades, and generate performance reports.

## 🚀 Features
- Add student details
- Input and manage marks
- Calculate average score
- Identify highest and lowest marks
- Display performance summary

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Arrays / ArrayList

## ▶ How to Run
1. Compile the program:
   javac StudentGradeTracker.java

2. Run the program:
   java StudentGradeTracker

## 📊 Output
The program displays student performance reports based on input marks.
